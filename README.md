# Sistem Informasi Kampus UAS Pemrograman Berorientasi Objek

Proyek ini adalah tugas akhir semester 3 (UAS) yang mengimplementasikan sistem manajemen kampus sederhana menggunakan bahasa pemrograman Python.

## Fitur Utama
* **Registrasi Mahasiswa**: Pendaftaran ID mahasiswa baru.
* **Modul Presensi**: Simulasi kehadiran menggunakan ID (QR Code) dengan timestamp.
* **Modul Keuangan**: Pencatatan pembayaran SPP dan penghitungan sisa tunggakan.
* **Modul Nilai & IPK**: Input nilai akademik dan penghitungan IPK otomatis.
* **Modul Jadwal**: Informasi mata kuliah dan ruang kelas.

## Teknologi yang Digunakan
* **Bahasa**: Python 3.x
* **Library**: `datetime` (untuk sistem waktu presensi)
* **Dokumentasi UML**: Mermaid.js
